Register and Download the SDK package

The first thing is to sign up in Xunfei Open Platform. Then enter into the personal application, there you can create an application and add the service you need(you can choose SDK or WebAPI or other formats, in this case, I used Linux SDK). After that, download the SDK package.

The official website has very clear instructions, check the website to see more details of downloading SDK. Here will be no more explanation.

Prepare the data

The sample code Xunfei offers recognizes one voice a time. But what we need is to process a batch of voices. To reserve all the filenames of wave files in the specified directory, use the ‘getfilename.c’ to save all the filenames in a txt file. (I didn’t use array because it’s too complicated to maintain an dynamic string array in C language.) And you can use ‘sort.cpp’ to sort the filenames. I wrote this simply for the convenience of my own using, it’s not necessary.

To use the SDK

After unzipping the SDK package, create an new folder named “demo”(just an example).

Transfer the directory ‘/bin’,’/include’,’/libs’ in sdk to the new folder.

Create ‘demo.c’ under the directory.(the code is in the repo, and don’t forget to change the APPid!)

Create the Makefile.Make sure the directory and target are right.

#common makefile header
DIR_INC = ./include
DIR_BIN = ./bin
DIR_LIB = ./libs

TARGET = demo
BIN_TARGET = $(DIR_BIN)/$(TARGET)

CROSS_COMPILE = 
CFLAGS = -G -Wall -I$(DIR_INC)

ifdef LINUX64
LDFLAGS := -L$(DIR_LIB)/x64
else
LDFLAGS := -L$(DIR_LIB)/x86

OBJECTS := $(patsubst %.c,%.o,$(wildcard *.c）

$(BIN_TARGET) : $(OBJECTS)
       $(CROSS_COMPLIE)gcc $(CFLAGS) $^ -O $@ $(LDFLAGS)

%.o : %.c
       $(CROSS_COMPILE)gcc -c $(CFLAGS) $< -O $@
clean:
       @rm -f *.o $(BIN_TARGET)
.PHONY:clean

Transfer the 64bit_make.sh to demo directory. (It also has the 32bit ver.)

‘cd demo’, then ‘source 64bit_make.sh’

‘cd bin’, run the target file ‘demo’.

Error handling

There will be only one error when using the code above, it’s error ‘10114’. It means session timeout. Just need to check the network connections or change a network.

Limitation

The Xunfei SDK only offer 500 times service for free everyday.
